# gulp-getting-started

1. Clone project
2. Install project dependencies
   * Install node

     ```
     $ sudo apt-get update
     $ sudo apt-get install nodejs
     ```
   * Install node packages

     ```
     $ npm install
     ```
3. Done with installation.

# How to use
   * To run local server

     ```
     gulp server
     ```
   * To create build

     ```
     gulp build
